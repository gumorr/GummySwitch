# GummySwitch

1 MB ROM switch for Amigas with 1 MB ROM mods (can also work with 2 MB ROMs)

![pcb](docs/pcb.png?raw=True)

## Use

Position 1-2 on the jumper selects ROM 2 (U4), while position 2-3 selects ROM 1 (U3)
